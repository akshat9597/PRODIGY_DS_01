# 📊 Data Visualization Project (EDA)
## 📌 Project Overview
This project demonstrates Exploratory Data Analysis (EDA)
using Python.

Random employee data was generated and visualized
using different plots.

## 📊 Visualizations Created

- Histogram (Age Distribution)
- Histogram (Salary Distribution)
- Bar Chart (Gender Distribution)
- Pie Chart (Department Distribution)

## 🛠 Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📁 Dataset

Randomly generated dataset containing:

- Age
- Gender
- Salary
- Department

## ▶ How to Run

1. Open the notebook
2. Run all cells
3. View visualizations
